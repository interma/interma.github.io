# How to build ORCA

## gpos

    mkdir build
    cd build
    cmake -D CMAKE_BUILD_TYPE=DEBUG ../
    make
    sudo make install

## gp-xerces

    mkdir build
    cd build
    env CFLAGS="-g" CXXFLAGS="-g" ../configure --prefix=/usr/local
    make
    sudo make install

## gp-orca

    mkdir build
    cd build
    cmake -D CMAKE_BUILD_TYPE=DEBUG ../
    make
    sudo make install
